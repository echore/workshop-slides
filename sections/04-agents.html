Part 4 · Agents

What is Agent

Given a goal, it plans its own steps, uses tools, and keeps adjusting until it's done.

Goal

You give it a goal, not a single instruction.

Tools

It can call outside tools, files, and systems to actually act.

Loop

After each step it checks the result and decides what to do next.

A fixed set of steps that never changes is automation. An agent checks and adjusts as it goes.

Source: Anthropic, “Building Effective Agents”

Generative AI vs Agentic AI

Same foundation, different level of autonomy.

Generative AI Agentic AI
What it does
Mainly generates content, such as text, images, or code.
Uses a generative model to plan, take actions, and adjust based on results.
How you interact
You ask for a response.
You give it a goal. It breaks the work into steps and continues toward that goal.
How it uses tools
It may use tools in a predefined workflow.
It can dynamically select tools and decide the next step based on results.
What's underneath
A generative model.
A generative model plus an action and feedback loop.

How agent works

It repeats this loop until the goal is done.

Perceive

Takes in information: files, data, tool results.

→
Reason

Makes sense of it and figures out what's relevant.

→
Decide

Picks the best next action among the options.

→
Act

Carries it out, often by calling a tool.

→
Check & adjust

Reviews the result and adjusts, then repeats.

The initial goal is given at the start. The agent may pause for input or refine its plan as it works.

Source: IBM, “How agentic AI works”

Example

It did the analyst's job, start to finish.

The data: monthly hospital admissions for acute coronary events in Sicily, before and after Italy's 2005 public smoking ban.

We asked: “Was the ban associated with fewer admissions? Fit a model, check the assumptions, address any problems, and explain it plainly.”

Step 1Inspected the data and selected an appropriate model.
Step 2Fit the baseline model.
Step 3Caught a problem: autocorrelation in the residuals.
Step 4Adjusted for seasonality, refit, and re-checked.
ResultModel estimate: ~11.5% lower rate; ~2,560 fewer admissions than the no-ban counterfactual.
